# DP

## J
`dp[i, j, k]` : i, j, k をそれぞれ、残り 3 個、2 個、1 個の皿の数としたときの期待値

とすると、残り 0 個の皿の数は `n - i - j - k` であり、漸化式は、
```
dp[i, j, k] = 1 + ((n - i - j - k) * dp[i, j, k]
	+ i * dp[i - 1, j + 1, k]
	+ j * dp[i, j - 1, k + 1]
	+ k * dp[i, j, k - 1]) / n;
```

## M
`dp[i, j]` : 先頭の i 人で j 個の飴を分ける方法の数の、0 から j までの累積和

## N
2つの部分に分割してそれぞれのコストの最小値を求め、その2つの和の最小値。  
トーナメント方式。
